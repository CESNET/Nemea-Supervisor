Detailed Description

Contains internal structures of all available modules and instances, functions for allocation/release of internal structures and also dynamic arrays that contain lists of available modules and instances that are part of the running configuration.

◆ specific_ifc_params_t

typedef union specific_ifc_params_u specific_ifc_params_t

Parameters specific to interface of given type.

This union holds parameters specific to interface of given type. Only one set of parameters can be stored though there is no distinction between IN and OUT interface types. Parameters specific for IN or OUT are merged inside structured linked by this union.
