◆ WAIT_FOR_INSTS_TO_HANDLE_SIGINT

#define WAIT_FOR_INSTS_TO_HANDLE_SIGINT   500000

Time in micro seconds between sending SIGINT and SIGKILL to running modules_ll. Service thread sends SIGINT to stop running module, after time defined by this constant it checks modules_ll status and if the module is still running, service thread sends SIGKILL to stop it.
