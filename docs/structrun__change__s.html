Detailed Description

Specifies change from sysrepo, node that got changed, type and action to take.

Type and names gets parsed from XPATH of changed element. In case type is NS_CHE_TYPE_GROUP_NODE or NS_CHE_TYPE_GROUP_ROOT, module_name is NULL.
